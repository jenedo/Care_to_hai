## Step 2 — Backend first: Add POST /api/auth/patient/register

Add to artifacts/api-server/src/routes/auth.ts:

POST /auth/patient/register
Body: {
  fullName: string,
  email: string,
  password: string,
  phone: string
}

Logic:
- Validate all fields present
- Check email not already in usersTable
- bcrypt.hash(password, 12)
- Insert into usersTable: { email, passwordHash, role: "PATIENT", status: "ACTIVE" }
- Insert into patientsTable: { userId, fullName, email, phone }
- Auto-login: sign JWT same as patient login
- Set httpOnly cookie "patient_token"
- Return: { success: true, data: { token, patient: { id, email, fullName, phone } } }
- If email exists: 409 { success: false, error: { code: "EMAIL_EXISTS", message: "Account already exists" } }

Test with curl before moving to frontend:
curl -X POST http://localhost:3000/api/auth/patient/register \
  -H "Content-Type: application/json" \
  -d '{"fullName":"Test Patient","email":"test@patient.com","password":"Test@1234","phone":"+92-3001234567"}' \
  -s

Expected: HTTP 201, token, patient object
