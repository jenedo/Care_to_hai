## AuthContext.tsx — Patient Auth

```typescript
// Pattern: EXACTLY like doctor-app AuthContext
// Changes:
const API_BASE = process.env.EXPO_PUBLIC_API_URL ?? "http://localhost:3000"

// PatientUser type:
type PatientUser = {
  id: string
  email: string
  fullName: string
  phone: string | null
  avatarUrl: string | null
}

// Login: POST /api/auth/patient/login
// Me: GET /api/auth/patient/me  
// Logout: POST /api/auth/patient/logout
// Token storage: AsyncStorage key "patient_token"
// Session restore on mount: Bearer token in Authorization header
```

## Components

### AppointmentCard.tsx
Props: appointment object, onPress
Shows: doctor name, date, status badge, type icon

### DoctorCard.tsx  
Props: doctor object, onPress
Shows: avatar initials, name, specialty, city, rating, fee

### StatCard.tsx
Props: title, value, icon, color
Shows: colored icon card with number

### Badge.tsx
Props: status: "confirmed"|"completed"|"cancelled"|"pending"|"verified"
Colors:
- confirmed → blue
- completed → green
- cancelled → red
- pending → yellow
- verified → green

### LoadingSpinner.tsx
- Centered sky blue spinner
- Optional message prop
